# Calculus Final Exams Table

A simple HTML project built as part of my coding practice.

## About the Project

This project is a basic HTML webpage that displays a table of calculus final exam grades. It includes student names, individual grades, and the average grade shown in the table footer.

The page was created to practise building structured tables in HTML and organizing tabular data clearly.

## Features

- Table with a caption
- Table header, body, and footer sections
- Student first names and last names
- Individual exam grades
- Average grade displayed in the footer
- Clear and simple tabular layout

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Building tables with the `table` element
- Using `caption` to describe table content
- Structuring table data with `thead`, `tbody`, and `tfoot`
- Creating rows with `tr`
- Adding header cells with `th`
- Adding data cells with `td`
- Using `colspan` to merge cells
